# Book Shelf Mobile

## Installation

### Fisrt
Clone and enter the repo

```bash
git clone https://github.com/matheuscosta2204/bookShelfMobile.git && cd bookShelfMobile
```
### Second
Install the node_modules repository

```bash
npm i
```

## Running

To run the project

```bash
react-native run-ios or react-native run-android
```
